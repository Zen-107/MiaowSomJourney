# 🐱 Miaow Som Journey - เกมการศึกษาเกี่ยวกับทฤษฎีกราฟ

## 📖 บทนำ

**Miaow Som Journey** เป็นเกมการศึกษาแบบ 2D ที่พัฒนาด้วย **Godot Engine 4.6** เพื่อช่วยให้ผู้เรียนเข้าใจแนวคิดพื้นฐานของ **ทฤษฎีกราฟ (Graph Theory)** ผ่านการเล่นเกมแบบโต้ตอบ

เกมนี้นำเสนอ Adjacency Matrix, Depth-First Search (DFS), Graph Traversal และ Spanning Tree ผ่านการผจญภัยของตัวละครแมวตัวจ้อยที่เดินทางข้ามต่างพื้นที่และต่อสู้กับสัตว์ประหลาด

---

## 🎮 ลักษณะเด่นของเกม

### ✨ คุณสมบัติหลัก

1. **ระบบเรียนรู้แบบเชิงปฏิบัติ**: ผู้เล่นเรียนรู้แนวคิดทฤษฎีกราฟผ่านการเล่นเกม
2. **ระบบ HP และไอเทม**: ตัวละครมี HP และสามารถเก็บไอเทมเพื่อช่วยเหลือ
3. **ระบบต่อสู้กับสัตว์ประหลาด**: พบปะกับศัตรูต่างๆ และต้องผ่านการต่อสู้
4. **ระบบ DFS และ Spanning Tree**: เกมติดตามเส้นทางการสำรวจและแสดงผล Spanning Tree
5. **โหมดหมายเหตุ (Hint System)**: ระบบบอกใบ้เพื่อช่วยผู้เล่น
6. **ระบบคะแนน**: ติดตามการสำรวจและให้คะแนนตามการเดินทาง

---

## 🕹️ วิธีการเล่น

### สไตล์การเล่น

เกมใช้ระบบ **Graph Navigation** ที่ผู้เล่นต้อง:
- ไปเยี่ยมชมตำแหน่งต่างๆ ที่เชื่อมกันด้วยขอบ (edges)
- ทำการสำรวจให้ครบทุกโหนด (nodes)
- ใช้กลยุทธ์ DFS ในการเยี่ยมชมโหนด
- และต้องต่างเจ้าหญิงปลาทูให้พบ 

### ระบบไอเทม

ผู้เล่นสามารถเก็บและใช้ไอเทมต่อไปนี้:

| ไอเทม | รายละเอียด | ประโยชน์ |
|------|----------|--------|
| 🥤 น้ำวิเศษ | เครื่องดื่มหวานปนขม | เพิ่ม HP +1 |
| 🛡️ เกราะแดง | เกราะสีแดงงดงาม | ป้องกันด้วย +1 HP |
| ⚔️ ดาบขี้เรืองแสง | ดาบที่มีแสงพิเศษ | ทำให้สัตว์ประหลาดหลับตาได้ |

### ระบบสัตว์ประหลาด

มีสัตว์ประหลาดหลายตัวที่จะปรากฏในการเดินทาง:
- กากเบี้ยว
- กีกี้คนสวย
- นุ่ยนุ้ยนาย
- และอื่นๆ

## 📚 แนวคิดการศึกษา

เกมนี้ประยุกต์ใช้เนื้อหาจากบทเรียน CPE101 ดังนี้:

### 1. **Adjacency Matrix** 🔢
   - แสดงเป็นรหัสตัวเลข 0 และ 1
   - บอกการเชื่อมต่อระหว่างตำแหน่งต่างๆ
   - ใช้ในระบบ Hint เพื่อแนะนำเส้นทาง

### 2. **Graph Representation** 📊
   - Undirected Connected Simple Graph
   - ทุกโหนดเชื่อมต่อกันไม่มีทิศทาง
   - ขอบเพียงมีค่า 1 หรือ 0 เท่านั้น

### 3. **Depth-First Search (DFS)** 🔍
   - วิธีการสำรวจเส้นทางไปให้ลึกที่สุด
   - ตัวละครจะเดินตามเส้นทาง DFS
   - กลับไปที่จุดแยกเมื่อไปถึงปลายทาง

### 4. **Spanning Tree** 🌳
   - สรุปเส้นทางการเดินทางของผู้เล่น
   - แสดงผลลัพธ์หลังสิ้นสุดเกม
   - ใช้เพื่อแสดงว่าผู้เล่นเดินทางผ่านโหนดใดบ้าง

### 5. **Graph Traversal Analysis** 📈
   - ต้องสำรวจให้ครบทุกจุด 100%
   - เพื่อให้ได้ Spanning Tree ที่สมบูรณ์

## 🚀 ฟีเจอร์พิเศษ

### ระบบใบ้ (Hint System)
- ให้ผู้เล่นดู Adjacency Matrix
- แสดงการเชื่อมต่อระหว่างโหนด
- ช่วยให้ตัดสินใจเรื่องเส้นทาง

### ระบบต่อสู้
- ต่อสู้กับสัตว์ประหลาดเมื่อไปถึงโหนดบางจุด
- ใช้ไอเทมเพื่อป้องกันหรือเอาชนะ
- สตรีกหา: สามารถสู้ชนะแต่ละตัวได้เพียงครั้งเดียว

### ระบบไอเทม
- สามารถเก็บได้สูงสุด 3 ไอเทม
- ใช้เพื่อเพิ่ม HP หรือป้องกันความเสียหาย
- บางไอเทมมีประสิทธิพิเศษ

### Map Maker (ในส่วน More Menu)
- ผู้ใช้สามารถสร้าง Adjacency Matrix เอง
- แปลงเป็น Graph visualization
- ทำความเข้าใจการออกแบบ Graph
